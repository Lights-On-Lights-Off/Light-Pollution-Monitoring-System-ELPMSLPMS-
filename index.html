NBSC Light Pollution Monitoring System

Visualizing Campus Light Pollution for Better Management

About the System

The NBSC Light Pollution Monitoring System is a simple web-based tool that visually presents light pollution levels within the Northern Bukidnon State College campus using a color-coded map.

Purpose

This system aims to raise awareness of excessive lighting and support better campus lighting management through visual data representation.
